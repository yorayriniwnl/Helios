# Team Roles — Who Built Helios

Backend
- Built by the backend engineering team: APIs, ingestion pipelines, processing workers, database models, authentication, and observability.
- Delivered: production‑ready FastAPI services, robust ingestion and seeding scripts, DB schema and tests, containerization and deployment manifests.

ML
- Built by the machine‑learning team: data engineering, feature pipelines, model training, evaluation, deployment and retraining workflows.
- Delivered: feature extraction pipelines, ensemble scoring models with explained confidence, model registry and a human‑in‑the‑loop retraining process.

Frontend
- Built by the frontend engineering team: operator UX, dashboards, real‑time feeds and the inspector evidence capture flow.
- Delivered: Next.js + Tailwind dashboards, action‑first interfaces (Action Queue, Zone Priority Map), and mobile‑friendly inspector views.

Summary
- This product was built by a focused, cross‑functional engineering team: Backend, ML and Frontend collaborated closely to deliver a demo‑ready, pilot‑grade system that is production‑ready to run city pilots and scale.
