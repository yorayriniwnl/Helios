# Memory Hook — Visual + Phrase

Visual (single image):
- A bright Helios sunbeam acts as a spotlight over one electricity meter; the beam reveals a glowing currency coin (₹/$) and a padlock that snaps shut on the meter — a subtle lens‑flare sweep animates once on appearance. Clean, high‑contrast iconography so it reads at a glance.

Phrase (3 words, rhythmical):
- Spot. Seal. Recover.

Why it sticks:
- Three short verbs (crisp cadence) make it easy to memorize and repeat.
- The visual ties the brand (sun/Helios) to the action (find → secure → reclaim value).

How to use:
- Final slide headline, spoken as the closing line with a 0.5s pause between words.
- Slide footer or corner watermark (small animated SVG: sun beam + lock) on all investor/demo slides.
- Stickers, one‑pager header and speaker notes: use the phrase as the mic-drop line.

Suggested spoken close (10s):
"Spot. Seal. Recover. That's Helios — we find the theft you can fix, reclaim revenue fast."
