Final Practice Loop — Step-by-step routine

Goal: run the full demo end‑to‑end until it is smooth — defined as two consecutive full runs with clean timing, no filler words, and reliable fallback behavior.

Routine (repeat loop until smooth):

1) Warm up (3–5 min)
- Breathing and posture: inhale 1s, exhale. Run the opening line twice with the voice modulation guide.
- Quick tech check: open `?demo=silent` or `docs/silent_demo_play.html` and confirm the backup plays.

2) Full timed run (recorded) — do the whole demo start → close, exactly as on stage (10–90s depending on demo length).
- Use a visible timer and record the screen.
- Immediately note 3 quick bullets: timing, one rough stumble, one tech note.

3) Quick playback review (3–6 min)
- Watch the recording at 1x, mark one fix per minute of demo. Apply the top 1–2 fixes.

4) Repeat full run (2 more times)
- Focus each run on one improvement: run A = voice/pauses, run B = eye contact/transitions.

5) Micro‑runs (10–15 min)
- 3–5 reps of the opener, remediation card, and close (separately). Use the voice and eye contact guides.

6) Q&A drills (10–15 min)
- Rapid-fire: 10 short questions. Use the calm_under_pressure script: restate, give headline, one supporting fact, next step.

7) Failure drills (5–10 min)
- Simulate a demo break twice: use the recovery line, switch to `docs/silent_demo_play.html`, and resume.

8) Final full runs
- Do final recorded runs until you get two consecutive "smooth" runs.

Definition of "smooth" (must meet all):
- Within target time (+/- 10%).
- No filler words in headline or close.
- Pauses and emphasis used correctly on opener and close.
- Eye contact transitions feel natural (see `docs/eye_contact_strategy.md`).
- Backup fallback (silent demo or recording) works and is ready.

Post‑run checklist (quick):
- Timing OK / notes
- Top 1 fix applied
- Recording saved and labeled
- Backup file tested

Practical cadence suggestions:
- Solo prep: 30–60 minutes total (1 warmup + 3 full runs + micro + Q&A + failure drills).
- Team dress rehearsal: 60–90 minutes with one person running, one moderating, one playback/fixer.

Stop condition: two consecutive full runs that pass the "smooth" checklist. If not achieved, iterate focused runs on the weakest area.

Repeat until smooth.
